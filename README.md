<h1>Hangman in Java</h1>

<h2>Description</h2>
A simple version of the classic game "Hangman" using Java Swing for it's UI. The game gets a list of random words from an API(dictionaryapi.dev) and their definition(to serve as an hint), and prompts the user to guess letters untill he either guesses correctly or runs out of tries.
Once the game is finished the user can input his name to save his score in a .txt file.
<br />


<h2>Languages and Utilities Used</h2>

- <b>Java</b> 
- <b>Swing</b>
